# 🧠 Equation Solver CLI

A simple Python command-line tool that solves linear and quadratic equations.

I built this to practice Python and apply some math without doing everything by hand.

---

## 🧠 What it does

* Solves **linear equations**:

  ax + b = 0

* Solves **quadratic equations**:

  ax² + bx + c = 0

* Handles different cases:

  * Two solutions
  * One solution
  * No real solution

---

## ⚙️ Features

* Simple CLI (runs in the terminal)
* Input validation (so it doesn’t crash on bad input)
* Clean and readable output
* Handles edge cases like `a = 0`

---

## ▶️ How to run

1. Make sure Python 3 is installed
2. Run the script:

```bash
python equation_solver.py
```

3. Choose what you want to solve and enter the values
